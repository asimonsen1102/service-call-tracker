ENTERPRISE FEATURES
✔ Role-based access (Admin / Dispatcher / Tech)
✔ Photo uploads (Firebase Storage)
✔ Push notifications (Firebase Cloud Messaging)
✔ Google Calendar auto-booking (Cloud Functions)

SETUP STEPS
1. Firebase project
2. Enable Auth, Firestore, Storage, FCM
3. Create users collection with role field
4. Add Cloud Functions for:
   - Push notifications on new calls
   - Google Calendar event creation
5. Paste Firebase keys into app.js

ANDROID
- Works as PWA or Capacitor APK
